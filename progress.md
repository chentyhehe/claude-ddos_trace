### 2026-04-14 - 告警驱动分析模式实现

**时间**：2026-04-14

**需求变更**：

- 需要从告警表 `detect_attack_dist` 自动获取阈值、时间窗口和攻击类型

**实现内容**：
1. [x] 创建 `AlertLoader` 类（`data/alert_loader.py`）— 从 `detect_attack_dist` 加载告警

3. [x] 重写 `DDoSTracebackAnalyzer` — 支持 3 种入口模式
4. [x] 更新 `api.py` — 新增 `/api/v1/analyze/alert` 和 `/api/v1/analyze/target` 端点
5. [x] 更新 `config.yaml` — 添加 `alert_table_name` 配置
